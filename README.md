# The Idea

A chrome extension to add custom features to the ChatGPT interface.

![Showcase](showcase.mov)

## Features

- Add code blocks to the message input
  - [x] Python
  - [x] Typescript
  - [x] HTML
  - [x] Markdown
  - [x] Javascript
  - [x] C++
  - [x] C
  - [x] Java
  - [] C#
  - [] Go
  - [] Rust
  - [] Ruby
  - [] PHP

- Add custom prompts
  - [] prompts list
  - mange prompts
    - [] add
    - [] remove
    - [] edit

- [] Add history to private chats
